# Извлечение информации из изображений  

## Описание  

Веб-приложение предназначено для анализа графических файлов и отображения основных характеристик изображений.  

Приложение показывает следующие данные:  
- Имя файла  
- Размер изображения (в пикселях)  
- Разрешение (DPI)  
- Глубина цвета  
- Метод сжатия (для поддерживаемых форматов)  

Приложение работает полностью в браузере и не требует установки.  

---

## Возможности  

- Поддержка форматов: **JPG, JPEG, GIF, TIF, TIFF, BMP, PNG, PCX**  
- Загрузка целой папки с изображениями (через `<input type="file" webkitdirectory multiple>`)  
- Обработка большого количества файлов (до 100 000)  
- Динамическое построение таблицы с результатами анализа  
- Индикатор прогресса обработки файлов  
- Красивый и удобный интерфейс (адаптивный дизайн)  
- Полностью клиентское приложение (не требует серверной части)  

---

## Технологии  

- **HTML5** — структура страницы  
- **CSS3** — встроенные стили оформления  
- **JavaScript (ES6+)** — встроенная логика обработки файлов  
- **File API** — работа с локальными файлами пользователя  
- **Canvas API** — анализ глубины цвета  
- **DataView** — чтение заголовков файлов для извлечения DPI, битности и метода сжатия  
- **URL.createObjectURL** — эффективная загрузка изображений  

---

## Как это работает  

1. **Выбор папки с изображениями**  
   Пользователь выбирает директорию с помощью поля ввода. Приложение получает список всех файлов внутри.  

2. **Фильтрация файлов**  
   Берутся только файлы поддерживаемых форматов (JPG, JPEG, GIF, TIF, TIFF, BMP, PNG, PCX).  

3. **Извлечение метаданных**  
   - **Имя файла** — из объекта `File`  
   - **Размер (px)** — через объект `Image` (ширина × высота)  
   - **Разрешение (DPI)** — из заголовков файлов (JPEG — JFIF/EXIF, PNG — `pHYs`, BMP/TIFF/PCX — поля заголовка). Если данных нет, выводится «—»  
   - **Глубина цвета** — определяется по формату (например, JPEG 24 бит, GIF 8 бит, PNG до 16 бит)  
   - **Метод сжатия** — определяется по типу файла (JPEG, Deflate для PNG, LZW для GIF, RLE/None для BMP и т.д.)  

4. **Отображение данных**  
   Все результаты отображаются в таблице. При наведении строки подсвечиваются.  

5. **Прогресс-бар**  
   При анализе большой папки отображается прогресс обработки (в процентах).  

---

## Пример использования  

1. Открыть HTML-файл в браузере  
2. Выбрать папку с изображениями  
3. Дождаться обработки файлов  
4. Просмотреть результаты в таблице
